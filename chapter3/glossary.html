HTML and CSS Glossary

Chapter 1

HTML Element
An HTML element, also known as a tag, is code placed in the .html file that either structures the web page such as <!DOCTYPE>, <html>, <head>, and <body> or describes specific content on the page such as <h1>, <p>, <a>, <ul>, and <li>. Some HTML elements are called "wireframe" elements in that they describe large areas of the web page such as <header>, <nav>, <main>, and <footer>.
Attribute
An HTML attribute is additional information placed in the opening tag of an HTML element that further describes the element. For example, the <a> anchor element needs information to identify the specific hyperlink. The href attribute is used to do that. For example, <a href="http://www.jccc.edu">JCCC</a> links the text JCCC to the home page for JCCC found at www.jccc.edu.
DTD
Document Type Definition identifies the version of HTML in the document.
Head Element
The head element contains the head section of the document. It begins with the <head> tag and ends with the </head> tag. The title element and the meta element are coded inside the head element.
Title Element
The title element is the first element in the head section. It configures the text that appears in the title bar of the browser window. The title element begins with <title> tag and ends with the </title>tag.
Body Section
The body section contains text and elements that display directly on the web page within the browser window.

Chapter 2

Nav Element
The purpose of the HTML5 nav element is to contain a section of navigation links. It begins with the <nav> tag and ends with the </nav> tag.
Absolute Hyperlink
Absolute hyperlinks indicate the absolute location of a resource on the web.
Line Break Element
The line break element makes the browser advance to the next line before displaying the next element or text on a web page. It is a void element that is coded as <br>.

Chapter 3

Image Optimization
Image optimization is the process of creating an image with the lowest file size that still renders a good-quality image — balancing image quality and file size.
Monochromatic Color Scheme
Monochromatic Color Schemes consist of shades, tints, or tones of the same color.
Perceived Load Time
Perceived Load Time is the amount of time your web page visitor is aware of waiting while your page loads.

Chapter 4

Selector
CSS selectors are used to find or select the HTML elements you want to style. It is the first part of a CSS Style rule. There are many different types of selectors. They can be an HTML element name, a class name, or an id name. A class selector begins with a period then class name. An id selector begins with a number sign then the ids name. An example of a selector is: body{declaration: value;}. The "body" in the example is the selector.
Declaration
The declaration indicates the CSS property you're setting and the value you're assigning to the property. An example is: property: value;
Property
The CSS property dictates what aspect of the selector will be changed visually.
The background-color Property
A CSS property to configure the background color of an element.
The color Property
A CSS property to configure the text color of an element.
Link Element
The link element associates an external style sheet with a web page. It is placed in the head section of the web page. It is also a void tag that uses the rel and href attributes to tell the web page the relationship, which would be "stylesheet", and the name of the style sheet file. Here's an example of a link element: <link rel="stylesheet" href="color.css">

Chapter 5

JPEG
Joint Photographic Experts Group (JPEG), is best used for photographs. JPEG images can't be made transparent or animated, but they support up to 16.7 million colors. They usually have .jpg or .jpeg file extensions. JPEG images are saved using lossy compression. This means that some pixels are lost or removed from the compressed file.
The font-weight Property
Configures the boldness of text. font-weight: bold; is similar to the <strong> or <b> HTML elements.
The text-decoration Property
Modifies the display of text. Commonly used values are none, underline, overline, and line-through. With this property you can edit the line under hyperlinks.

Chapter 6

The min-width Property
Sets the minimum width of an element's content in the browser viewport.
The max-width Property
Sets the maximum width of an element's content in the browser viewport.
The height Property
Configures the height of an element's content in the browser viewport with a numeric value, a percentage of the parent element, or viewport height value.

Chapter 7

The float Property
Floats an element to either the left or right side of the screen. It renders the element in normal flow and then shifts them to the left or right as far as possible in the browser view port.
The CSS display Property
Configures the way that browsers render elements. Its values are none, inline, inline-block, block, flex, and grid.
The :link Pseudo-class
The :link pseudo-class is one of five pseudo-classes that can be applied to the anchor element. :link is the default state for a hyperlink that has not been visited.

Chapter 8

grid-template-columns Property
The grid-template-columns property is used to configure the number of columns in a grid.
grid-template-rows Property
The grid-template-rows property is used to configure the number of rows in a grid.
align-items Property
The align-items property configures the way the browser displays extra space along the cross axis of the container. Its values are flex-start, flex-end, center, baseline, and stretch.

Chapter 9

The table Element
A block display element that contains tabular information. It begins with <table> and ends with </table>.
The rowspan Attribute
The rowspan attribute specifies the number of rows that a cell will occupy. An example is rowspan="2". This means that the cell spans two rows.
The colspan Attribute
The colspan attribute specifies the number of columns that a cell will occupy. An example is colspan="2". This means that the cell spans two columns.

Chapter 10

The form Element
The form element contains a form on a web page. It begins with <form> and ends with <form>. There can be multiple forms on a web page, but they cannot be nested inside of each other. The form element can be configured with attributes that specify what server-side program or file will process the form, how the form receives the information, and the form's name.
Form Controls
The purpose of a form control is to gather information from a web page's visitor. The form controls are the objects that accept the information. Types of form controls are text boxes, scrolling text boxes, select lists, radio buttons, check boxes, and buttons. HTML5 offers even more form controls like dates, times, URLs, numbers, and color section."
The Submit Button Form Control
The submit button form control is used to submit the a form on a website. When clicked it triggers the action attribute on the opening form tag, which causes the web browser to send the form data to the web server. The <input> tag with type="submit" configures a submit button.
