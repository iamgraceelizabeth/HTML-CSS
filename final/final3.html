Course Description

This website was made as a final project for the WEB 110 class given at Johnson County Community College.
The course requirements and outline are shown below.

WEB 110

  • Title: HTML and CSS
  • Number: WEB 110
  • Effective Term: 2021-22
  • Credit Hours: 3
  • Contact Hours: 3
  • Lecture Hours: 3

Description:

This course will cover the essential skills needed to create responsive websites, using Hypertext Markup Language (HTML) and Cascading Style Sheets (CSS). Students will be introduced to the concepts, foundations, syntax and structure of HTML and CSS. Additional topics include the use of File Transfer Protocol (FTP) to publish websites and validation to web standards established by the World Wide Web Consortium (W3C) and other organizations.

Objectives

  1. Explain how to create sites that are compliant with current W3C standards in web development.
  2. Create valid and semantically correct HTML pages.
  3. Organize website files and FTP them to a web server.
  4. Create valid HTML tables.
  5. Create absolute, relative and anchor links.
  6. Apply cascading styles to an HTML document through inline, internal or external style sheets.
  7. Test and troubleshoot HTML and CSS.
  8. Create web forms using HTML and style them with CSS.
  9. Apply responsive web design principles.

Content Outline and Competencies:

  1. Web Page Building Blocks
    • Develop web page files.
    • Describe W3C standards.
    • Discuss how Section 508 of the Americans with Disabilities Act affects web page design.
    • Write the source code of a web page.
    • Describe web page elements, attributes, and values.
  2. Structure and Content
    • Declare the encoding and create valid HTML structural and wireframe elements.
    • Create appropriate textual content.
    • Insert appropriate graphical content.
    • Mark up content with semantically correct HTML such as paragraphs, phrases, headings, lists, and images.
    • Code appropriate attribute values to further describe and control HTML elements.
  3. Web Page Files
    • Define and apply appropriate web file naming conventions and file management techniques.
    • Use FTP to deploy pages to a web server.
    • Test published web pages using HTTP protocol in a browser.
  4. Tables
    • Discuss the appropriate use for tables.
    • Create and modify table rows and columns.
    • Apply styles to tables.
  5. Absolute, Relative and Anchor Links
    • Create links.
      1. Relative links to other pages
      2. Absolute links to other websites
    • Create anchor links.
    • Create targeting links.
    • Use images as links.
    • Discuss image maps.
  6. Cascading Style Sheets
    • Define CSS selectors, attributes, and declarations.
    • Select elements by element name, type, or id, or class including combinators and pseudo-classes.
    • Create and appropriately apply external, internal, and inline styles.
    • Apply text styles such as font family, text color, background color, and alignment to both inline and block elements.
    • Apply styles using the box model such as borders, padding, and margins.
    • Create multi-column flexible layouts using both float and grid techniques.
    • Apply flexbox techniques.
  7. Testing and Validation
    • Test and validate HTML of the published website.
    • Test and validate CSS of the published website.
    • Employ popular browsers to test web pages.
    • Use browser developer tools to troubleshoot problems.
  8. Web Forms
    • Create a web form.
    • Create form styles using CSS.
    • Create form input elements and other controls used for a variety of data types.
    • Apply appropriate form element attributes to improve the accuracy and speed of data entry.
    • Explain the process of sending data from a web form to a server.
  9. Responsive Web Design
    • Define the goals and techniques of responsive web design.
    • Describe and apply flexible units of measurement for fonts and layout.
    • Create flexible images using CSS.
    • Create media queries in CSS.
    • Insert appropriate HTML meta tags for responsive design.
    • Apply both a traditional and mobile-first responsive style sheet.

Method of Evaluation and Competencies:

30-40% Homework and projects

30-40% Quizzes and tests

30-40% Final website

Total: 100%
